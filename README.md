# SQL-Analysis-Project-GlobalTech-Supplies
Este projeto simula uma empresa fictícia chamada GlobalTech Supplies, que atua na distribuição de produtos de tecnologia e escritório em diferentes regiões e segmentos de mercado. O objetivo é utilizar consultas SQL para analisar dados de clientes, pedidos, produtos e vendas, identificando gargalos de rentabilidade e apoiando decisões estratégicas.
